# Alicia-D SDK Examples

本目录包含了Alicia-D SDK的各种使用示例，展示了SDK的主要功能。

## 示例列表

### 基础示例

1. **01_basic_usage.py** - 基本使用示例
   - 展示SDK的基本功能
   - 连接、关节运动、笛卡尔运动、夹爪控制、在线控制

2. **02_advanced_control.py** - 高级控制示例
   - 展示高级控制功能
   - 轨迹规划、状态监控、错误处理、限位设置

3. **03_architecture_demo.py** - 架构演示示例
   - 展示分层架构设计
   - 各层组件的直接使用

### 功能示例

4. **04_demo_read_state.py** - 读取状态示例
   - 读取关节角度、末端位姿、夹爪状态
   - 实时状态监控

5. **05_demo_moveJ.py** - 关节空间运动示例
   - 单点关节运动
   - 多点关节轨迹
   - 弧度和度数格式

6. **06_demo_moveCartesian.py** - 笛卡尔空间运动示例
   - 单点位姿运动
   - 多点笛卡尔轨迹
   - 矩形、圆形、螺旋轨迹

7. **07_demo_gripper.py** - 夹爪控制示例
   - 基本夹爪控制
   - 角度控制
   - 夹爪与运动结合

8. **08_demo_torque_control.py** - 扭矩控制示例
   - 扭矩开关控制
   - 示教功能

9. **09_demo_zero_calibration.py** - 零点校准示例
   - 零点校准功能
   - 校准验证

10. **10_demo_online_control.py** - 在线控制示例
    - 实时在线控制
    - 复杂轨迹执行

## 使用方法

### 环境准备

1. 确保已安装Alicia-D SDK
2. 确保机械臂已连接并通电
3. 检查串口设置（默认COM6，波特率1000000）

### 运行示例

```bash
# 运行基本使用示例
python 01_basic_usage.py

# 运行特定功能示例
python 05_demo_moveJ.py

# 运行所有示例
for i in {01..10}; do python ${i}_demo_*.py; done
```

### 参数配置

大多数示例使用默认参数，如需修改：

```python
# 修改串口设置
robot = create_robot(port="COM3", baudrate=921600)

# 修改调试模式
robot = create_robot(port="COM6", debug_mode=False)
```

## 注意事项

1. **安全第一**: 运行示例前确保机械臂周围安全
2. **串口设置**: 根据实际情况修改串口和波特率
3. **限位检查**: 确保目标位置在机械臂工作范围内
4. **紧急停止**: 随时准备按Ctrl+C停止程序
5. **扭矩控制**: 关闭扭矩时注意安全

## 故障排除

### 连接问题
- 检查串口是否正确

- 检查硬件连接

### 运动问题
- 检查目标位置是否可达
- 检查关节限位
- 检查紧急停止状态

### 夹爪问题
- 检查夹爪连接
- 检查夹爪角度范围
